# 주석이란? (comment)
주석은 코드 내부에서 실행되지 않고 컴파일러가 무시하며 편의성을 위해 글이나 단어를 코드에 남기기 위해 존재한다.

## 주석을 코드에 추가하는 방법
```c
/* 주석 */

// 주석

/* 주석
주석
주석
주석
*/
```
위와 같이 남길 수 있고 보통 //는 주석이 한줄에 끝나는 경우 /* */는 주석이 여러줄에 걸쳐 사용하는 경우에 사용한다.

## 주석에 쓰임새
1. 코드를 쓸때 잠시 어떤 코드를 실행하고 싶지 않을때 사용한다. (에러가 났을때 어떤 코드가 원인인지 쉽게 파악 할 수 있음.)
2. 코드에 대한 설명을 남기기 위해 사용한다.

## 주석의 실용성
주석의 실용성을 느끼지 못할 수도 있다.
주석은 팀 단위로 코딩을 할때 동료가 코드를 읽을때 쉽게 읽을 수 있고,
1인으로 개발할때도 난해한 코드나 복잡한 코드를 작성할때 자신이 알아보기 편하기 위해서 주석을 사용할 수 있다.
