# "Hello, World!" 출력하기
```c
#include <stdio.h>
int main() {
  printf("Hello, World!");
  return 0;
}
```

위 코드를 컴파일하고 실행을 하면 Hello, World!가 출력되는 것을 볼 수 있다.

## "Hello, World!" 코드 분석

```c
#include <stdio.h>
```
위 코드는 stdio라는 표준 입출력(Standard Input Output)에 관련된 .h 파일을 불러오라는 뜻이다.
> ##### .h는 헤더파일을 뜻한다. 헤더파일은 첫번째로 언급한 코드의 printf 같은 명령어들이 담겨있는 상자이다.

---

```c
int main() {}
```
위 코드는 main이라는 함수를 선언하고 main함수가 다 실행되었을때 정수(int)형태의 값을 반환한다는 의미이다.
> ##### 함수가 아무리 여러개가 있어도 c로 작성된 프로그램은 main함수부터 실행된다. main함수는 오직 하나만 존재해야한다. (처음으로 실행되는 함수이기 때문)
> ##### 화면에는 Hello, World!가 출력되었지, 정수 형태의 숫자가 출력되진 않았다. 이는 return 0; 명령어에 의해 0이라는 정수가 반환되었기 때문에 정수가 반한된다고 표현한다. 참고로 0은 프로그램이 아무 문제 없이 잘 실행되었다고 알리는 신호이다.
> ##### main()에서 ()는 main이 함수라는 것을 표시하기 위해 사용되었고 {는 main함수의 명령어가 여기서부터라는 시작점을 의미하고 }는 main함수의 명령어의 끝을 의미한다.

---

```c
printf("Hello, World!");
```
위 코드는 Hello, World!를 출력하라는 의미이다
> ##### printf는 괄호 안에 있는 값을 출력하라는 의미이다. ""는 문자열을 출력하기 위해 따옴표 내부에 있는 값을 문자로 생각한다는 의미이다.
> ##### ;는 ; 앞에 있는 명령어가 끝남을 의미한다. 개별적으로 실행 가능한 모든 명령문 뒤에 붙여야 한다.

---

```c
return 0;
```
위 코드는 함수 내부에 있는 명령어들이 모두 정상적으로 실행되었다는 것을 의미한다. 정상적이지 않게 실행된 경우엔 0이 아닌 다른 숫자를 사용한다.
> ##### return 0;을 보면 어떻게 프로그램이 정상적으로 실행되었는지 미리 알 수 있는지에 대한 의문이 생긴다. 하지만 C는 1번째 줄부터 실행하기 때문에 에러가 나면 return 0; 명령어가 실행되지 못하므로 컴퓨터는 프로그램이 정상적이지 않게 실행된 걸 알 수 있다.
> ##### 나중에 배울 if문을 통해 0이 아니라 상황마다 다른 숫자를 넣으므로 프로그램을 자세하게 설계 가능하다.
