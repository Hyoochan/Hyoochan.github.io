# 컴파일러란? (Compiler)
컴퓨터는 전기가 흐름(1) 전기가 흐르지 않음(0)의 두 가지 상태로 여러가지 연산을 수행한다.
그래서 사실 컴퓨터가 이해하는 코드는 01010100101010101
같이 0과1로 이루어져있다. 그리고 이를 **기계어**라고 한다.
영화나 유튜브를 보다보면 개발자가 0과1만 사용하여 코드를 짜는 것은 볼 수가 없다.
> ###### 0과1을 사용하지 않는 이유는 당연하게도 가독성이 떨어지고 유지보수가 불가능에 가깝기에 그렇다.
개발자는 그래서 기계어 대신 **프로그래밍 언어**를 사용한다.
> ###### 우리가 아는 파이썬,C,Cpp,java등이 프로그래밍 언어에 포함된다.
프로그래밍 언어는,
```python
print(1+1)
```
과 같이 가독성이 기계어보다는 뛰어나다는 것을 알 수 있다.

하지만, 프로그래밍 언어엔 큰 단점이 있다. **컴퓨터가 이해를 못한다.**
그렇다면 개발자들은 어떻게 컴퓨터 언어를 사용하는 것일까?

바로 **컴파일러**라는 도구 때문이다.
컴파일러는 프로그래밍 언어를 기계어로 바꿔주는 역할을 수행한다.
